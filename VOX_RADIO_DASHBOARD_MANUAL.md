## Getting Started

### Accessing the Dashboard
1. Open your web browser
2. Navigate to the Vox Radio Dashboard URL
3. The dashboard will load with the **Current Weather** tab active by default

### Dashboard Layout
- **Top Bar**: Vox Radio branding and "Listen Live" button
- **Navigation Tabs**: Five main sections for different functions
- **Main Content Area**: Displays information and forms based on selected tab
- **Download Button**: Located in the top-right for generating reports

## Feature Sections

### Current Weather

**Purpose**: Provides real-time weather information for major Liberian cities to assist with weather-related programming and announcements.

#### What You'll See:
- **4 Weather Cards** displaying:
  - City name and country
  - Current temperature in Celsius
  - Weather condition (Sunny, Cloudy, etc.)
  - Humidity percentage
  - Wind speed in km/h
  - Last update timestamp

#### Cities Covered:
- **Monrovia, Liberia** (Capital city)
- **Gbarnga, Liberia** (Central region)
- **Buchanan, Liberia** (Coastal area)
- **Kakata, Liberia** (Western region)

#### How to Use:
1. Click on **Current Weather** tab
2. Review weather information for each city
3. Use this information for:
   - Weather announcements on-air
   - Planning outdoor events
   - Community safety alerts
   - Travel advisories

### WhatsApp Management

**Purpose**: Manage community engagement through WhatsApp messages sent to the station's business number (0777975975).

#### Features:

##### Statistics Dashboard
- **Today**: Number of messages received today
- **This Week**: Weekly message count
- **This Month**: Monthly message total
- **WhatsApp Number**: Display of the station's WhatsApp contact

##### Message Management
- **Recent Messages**: List of incoming WhatsApp messages
- **Status Tracking**: Shows which messages have been replied to
- **Quick Actions**: Reply and view details buttons

#### How to Use:

##### Viewing Messages:
1. Click on **WhatsApp** tab
2. Review statistics in the dashboard cards
3. Scroll down to see recent messages

##### Managing Messages:
1. **To Open WhatsApp**: Click the green "Open WhatsApp" button
2. **To Refresh Data**: Click the "Refresh" button
3. **To Add New Messages**: Use the "Add Message" button for manual entry

##### Best Practices:
- Check WhatsApp messages regularly (recommended: every 2 hours during broadcast)
- Respond promptly to listener queries
- Use the statistics to track community engagement trends

### Submit Testimony

**Purpose**: Collect and manage testimonies from community members for potential broadcast and community building.

#### Form Fields:

| Field | Required | Description |
|-------|----------|-------------|
| **Full Name** | Yes | Testimony submitter's complete name |
| **Phone Number** | Yes | Contact number (format: 0777123456) |
| **Testimony Category** | Yes | Type of testimony (see categories below) |
| **Your Testimony** | Yes | The actual testimony content |

#### Testimony Categories:
- **Healing & Recovery**: Health-related testimonies
- **Financial Breakthrough**: Financial blessings and improvements
- **Family Restoration**: Family unity and relationship healing
- **Spiritual Growth**: Faith and spiritual development
- **General Praise**: General thankfulness and praise
- **Answered Prayer**: Specific prayer responses

#### How to Use:

##### Submitting a Testimony:
1. Click on **Submit Testimony** tab
2. Fill in all required fields:
   - Enter the person's full name
   - Add their phone number
   - Select appropriate category from dropdown
   - Type or paste the testimony in the text area
3. Click **Submit Testimony** button
4. Confirmation message will appear upon successful submission

##### Managing Testimonies:
- **View Recent**: Check the right panel for recently submitted testimonies
- **Status Tracking**: See approval status (approved/pending)
- **Categories**: Use categories to organize testimonies for different programs

##### Best Practices:
- Verify caller information before submitting
- Choose the most appropriate category
- Keep testimonies concise but meaningful
- Follow up with submitters when testimonies are broadcast

### Log Phone Call

**Purpose**: Record and track all incoming phone calls for better listener engagement and follow-up management.

#### Call Logging Form:

| Field | Required | Description |
|-------|----------|-------------|
| **Caller Name** | Yes | Name of the caller (use "Anonymous" if not provided) |
| **Phone Number** | No | Caller's contact number |
| **Call Topic/Category** | Yes | Purpose of the call (see categories below) |
| **Presenter** | Yes | Staff member who took the call |
| **Call Duration** | No | Length of call (format: 5:30) |
| **Notes** | No | Additional details about the call |
| **Follow-up Required** | Checkbox | Mark if caller needs follow-up |

#### Call Categories:
- **Prayer Request**: Caller requesting prayer
- **Song Request**: Music requests for programming
- **Program Feedback**: Comments about shows
- **Complaint**: Issues or concerns raised
- **General Inquiry**: Information requests
- **Technical Issue**: Problems with reception/website
- **Community Announcement**: Events or news to share

#### Presenter Options:
- John Smith
- Sarah Wilson  
- Michael Johnson
- Grace Davis

#### How to Use:

##### Logging a Call:
1. Click on **Log a Phone Call** tab
2. Fill in the form during or immediately after the call:
   - Enter caller name (required)
   - Add phone number if provided
   - Select call category from dropdown
   - Choose your name as the presenter
   - Record call duration if noted
   - Add relevant notes
   - Check follow-up box if needed
3. Click **Log Phone Call** button

##### Managing Call Records:
- **View History**: Recent calls appear in the right panel
- **Follow-up Tracking**: Calls marked for follow-up are highlighted
- **Import Feature**: Use "Import Calls" for bulk entry

##### Best Practices:
- Log calls immediately while details are fresh
- Always categorize calls appropriately
- Use notes field for important details
- Mark follow-up for important issues
- Review follow-up calls regularly

### Recent Submissions

**Purpose**: Centralized view of all submissions (testimonies, phone calls, WhatsApp messages) for comprehensive management.

#### Features:

##### Filtering Options:
- **Search Bar**: Find submissions by name, content, or keywords
- **Type Filter**: Filter by submission type (All Types, Testimonies, Phone Calls, WhatsApp)
- **Status Filter**: Filter by approval status (All Status, Approved, Pending)

##### Submission Management:
- **View Details**: See complete submission information
- **Approval Actions**: Approve or reject pending submissions
- **Status Indicators**: Visual status badges for each submission

#### How to Use:

##### Viewing Submissions:
1. Click on **Recent Submissions** tab
2. Use filters to narrow down submissions:
   - Type in search bar for specific content
   - Select submission type from dropdown
   - Choose status filter if needed

##### Managing Submissions:
1. **To Approve**: Click the green checkmark button
2. **To Reject**: Click the red X button  
3. **To View Details**: Click the eye icon
4. **To Search**: Use the search bar for specific submissions

##### Best Practices:
- Review pending submissions regularly
- Use consistent approval criteria
- Keep approval/rejection reasons documented
- Monitor submission trends using filters

## Troubleshooting

### Common Issues:

#### Form Not Submitting:
**Problem**: Form shows error when submitting
**Solution**: 
1. Check all required fields are filled
2. Ensure phone numbers are in correct format (0777123456)
3. Refresh page and try again

#### Data Not Loading:
**Problem**: Sections appear blank or don't load
**Solution**:
1. Check internet connection
2. Refresh the browser page
3. Clear browser cache if needed

#### WhatsApp Button Not Working:
**Problem**: "Open WhatsApp" button doesn't respond
**Solution**:
1. Ensure WhatsApp is installed on device
2. Check if popup blockers are disabled
3. Try copying number (0777975975) manually to WhatsApp

#### Weather Information Not Updating:
**Problem**: Weather data appears outdated
**Solution**:
1. Weather updates automatically every 15 minutes
2. Refresh the page for latest data
3. Check timestamp on weather cards
